✖️ Multiplication Functions

Basic Function

function multiply(a, b) { return a * b; } console.log(multiply(5, 3)); // 15

Parameter Types

// Multiple parameters function multiplyThree(a, b, c) { return a * b * c; } // Default parameter function multiplyDefault(a, b = 1) { return a * b; } // Array parameter function multiplyArray(nums) { let result = 1; for (let num of nums) { result *= num; } return result; }

Real Example

function calculatePrice(price, qty, tax = 0.1) { return (price * qty) * (1 + tax); } console.log(calculatePrice(10, 2)); // 22
